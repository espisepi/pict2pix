Setting up Led Matrix Effect

  • Get the image element and call to pict2pix.animate function
    const imagejh = document.getElementById('image-jh');

window.onload = function initialize() {
    pict2pix.animate({
        image: imagejh,
        particleType: 'led-matrix',
        type: 'random',
        speed: 6, 
        transitionTime: 5000
    });
}
  • Options
    • image: Mandatory. The html element to apply the effect
    • particleType: The effect to apply to the image. In this case is 'led-matrix'
    • type: Mandatory. Transition effect type.
      • random
      • point
      • horizontal
      • vertical
    • speed: Optional. Transition effect speed. Range 1-10. Default value 8.
    • transitionTime: Optional. Time between transitions in ms. Default value 8000.
